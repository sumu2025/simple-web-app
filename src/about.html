项目简介

simple-web-app 是一个用于学习开发协作、git、docker、CI/CD等工具流程的简单项目。我们致力于通过实践来掌握现代软件开发的工具和流程。

  • 王五

项目目标和愿景

我们的目标是创建一个简单但功能完整的 Web 应用，通过这个项目：

  • 掌握 Git 协作流程
  • 学习 Docker 容器化技术
  • 实践 CI/CD 自动化部署
  • 体验现代软件开发的完整流程

技术栈

  • 前端：HTML, CSS, JavaScript
  • 后端：Node.js
  • 容器化：Docker
  • 版本控制：Git
  • CI/CD：GitHub Actions
